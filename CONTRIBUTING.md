## Development environments

### Developing on Eclipse

```
$ mkdir libsbmlsim-eclipse
$ ls
libsbmlsim-eclipse/ libsbmlsim-v2/

$ cd libsbmlsim-eclipse/
$ cmake -G "Eclipse CDT4 - Unix Makefiles" -DCMAKE_BUILD_TYPE=Debug ../libsbmlsim-v2
```
